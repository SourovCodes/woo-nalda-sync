Sync WooCommerce products to Nalda and receive orders from the Nalda marketplace.

== Description ==

Woo Nalda Sync allows WooCommerce store owners to sync their products to Nalda.com and receive orders from the Nalda marketplace.

= Features =

* Sync products to Nalda marketplace
* Receive and process orders from Nalda
* Automatic inventory synchronization
* Real-time order notifications
* Automatic plugin updates

= Requirements =

* WordPress 5.8 or higher
* WooCommerce 5.0 or higher
* PHP 7.4 or higher
* Valid license key

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woo-nalda-sync` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to WooCommerce > Nalda Sync to configure the plugin settings.
4. Enter your license key to activate the plugin.

== Frequently Asked Questions ==

= Where do I get a license key? =

You can purchase a license key from [jonakyds.com](https://jonakyds.com).

= How do I activate my license? =

Go to WooCommerce > Nalda Sync and enter your license key in the License section.

= Can I use the same license on multiple sites? =

No, each license is valid for one site only. You can deactivate your license from one site and activate it on another if needed.
